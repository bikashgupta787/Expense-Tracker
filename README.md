# SpendBhai 💸

**SpendBhai** is a modern Android expense tracking application built using **Kotlin, MVVM Architecture, Room Database, and Material Design 3**.
The app helps users manage daily expenses, track spending habits, set budgets, and analyze expenses with intuitive monthly insights and category-wise statistics.

---

# ✨ Features

- 📊 Track daily income and expenses
- 📅 Month-wise transaction filtering
- 💰 Budget management with progress indicators
- 📈 Category-wise expense analytics
- 🥧 Pie chart visualization for spending insights
- 🗂 Expense categorization with custom icons & colors
- 🔄 Swipe-to-delete with undo support
- 🧾 Clean and modern Material Design UI
- 📱 Fully offline support using Room Database
- 🌙 Optimized and responsive Android UI
